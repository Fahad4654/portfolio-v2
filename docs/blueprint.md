# **App Name**: BioFlow

## Core Features:

- Profile Display: Display personal information, including name, a profile picture, and a short bio.
- Link Showcase: Provide links to social media profiles and other relevant websites.
- Project Portfolio: Display a portfolio of projects with images and descriptions.
- Contact Form: Allow visitors to contact the user via email.
- Testimonials: Testimonials: Display quotes from clients or collaborators.

## Style Guidelines:

- Dark Slate Gray (#2F4F4F) for a calm and professional feel.
- Light Gray (#D3D3D3) for a soft and clean backdrop.
- Accent color: Steel Blue (#4682B4) to provide subtle contrast and highlight key interactive elements.
- 'Roboto' for a clean, modern, objective feel.
- Use simple, line-based icons for social media links and other actions.
- Design a clean and responsive layout that works well on all devices.
- Use subtle animations for transitions and hovers to provide feedback and enhance user experience.
